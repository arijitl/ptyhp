What is it?

Learner Profiling

A learner profile is basically information relating to individual users of the platform. Ptotemy offers a flexible tag profiling system such that the learners can be profiled in terms of demographics, needs or usage. This profiling system forms the basis of content targeting and analytics in Ptotemy.

Automated Need Analysis

Training Need Analysis is the essential first step of learning management where the learners' needs are identified. Ptotemy provides automated need analysis of the learners using the profiling system and can be used for automatic need based assignment of the learning content.

Ciphered Architecture

Ciphers are content keys which contextually have different values at different places for different users. Use Ptotemy's ciphering system to contextualize the content to the user in terms of language, customized examples, etc. Or just use it to manage frequently changing content from a central place.

Ontological Templates

An ontological structure represents knowledge as a hierarchy of concepts within a domain, using a shared vocabulary to denote the types, properties and interrelationships of those concepts. Ptotemy organizes your content into pre-structured ontological templates like Wikipedia for clearer comprehension.

Learning Flow Options

Learning Flows are the paths learners take to access content. Ptotemy allows multiple learning flows: Topic Flow for learners seeking detailed knowledge about a concept, Referential Flow for learners seeking answers to specific questions and Game Flow for learners browsing for knowledge as engagement.

What is it?

Flow Experience

Flow is the mental state in which a person performing an activity is fully immersed in a feeling of energized focus and enjoyment in the process of the activity. Ptotemy harnesses the power of Flow experience of Serious Games to induce higher engagement and retention of training content.

Discovery Learning

Practice in discovering for oneself teaches the learner to acquire information in a way that makes that information more readily viable in problem solving. Ptotemy provides games which encourage exploration and simulation as methodologies for learning by doing.

Spaced Repetition

Spaced repetition is a learning technique that incorporates increasing intervals of time between subsequent review of previously learned material in order to exploit the psychological spacing effect. Ptotemy Serious Games utilize simplicity and spaced repetition to reinforce the users' learning.

Gamification

Gamification is the process of using basic game mechanics like points and badges in a non-game context to influence behaviour. Ptotemy uses gamification specifically to encourage deeper perusal of the learning content and to capture data regarding the learners' interaction with the content.

Reward System

Motivational rewards go a long way in driving competition - collaboration and thereby, engagement with the learning content. Ptotemy's reward system can be used to create and assign rewards for content navigation and assessment performance by the learners.

What is it?

Parameterized Structures

All content in Ptotemy is taggable and cipherable. As a consequence of this, the data structures in Ptotemy are automatically parameterized, which allows for highly granular pivoting of the learner data for deeper analysis of the learner's behavior patterns.

Revalidated Data

Repetitive playing of games by the learner ensures that multiple iterations of data capture for a given context. Ptotemy provides aggregation and statistical analysis of the serious game data to provide baselining and revalidation of the data sets captured from the learner interactions.

Infographic Reports

Ptotemy provides data in both tabular and presentation-ready inforgraphic forms. The Reports Engine allows for dynamic creation of reports. The reports and the data can be exported into Excel or into PDF presentations complete with charts and data visualizations.

Real Time Data

Ptotemy is built using a live HTML framework. This allows real-time interaction of learner environments and collation of real-time data across learners. Ptotemy also provides dashboards which can be customized to display real time interaction data of the learners.

Content Quality Feedback

Ptotemy allows learners to provide detailed component level feedback to the content creator, indicating appreciation or lack of comprehension, requesting for additional explanation or references, etc. to encourage the creator to constantly improve the content.
